# SpectraSVD
Image compression using SVD (Singular Value Decomposition) in Python using NumPy, Pillow and Matplotlib.

## Libraries used:
- `streamlit`
- `Pillow`
- `numpy`
- `opencv-python`
- `imageio`
- `matplotlib`
